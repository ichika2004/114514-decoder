#  惡臭加密器

這是一個結合了 **RSA 非對稱加密演算法** 與 **114514 數字論證（Homo Encoding）** 的加密工具。它能將一段平凡的文字，轉化為由 `1, 4, 5` 組成的複雜數學算式，同時確保數據的安全性。

> 「這不僅僅是加密，這是一種藝術（與惡臭）。」

---

##  核心特色

-  **雙層防禦**：底層採用 RSA 非對稱加密，確保即便算式被還原，原始數據依據無法被破解。
-  **動態論證**：使用遞迴演算法（Recursive Decomposition）將密文動態拆解為 114514 算式。
-  **完美支援中文**：透過 UTF-8 與 Base64 轉換，完美解決中文字元加密後的編碼問題。

---

##  技術原理

惡臭加密流程遵循以下數學邏輯：

1. **RSA 加密層**：
   將明文 $M$ 透過公鑰 $(n, e)$ 進行加密：
   $$C \equiv M^e \pmod n$$
   得到密文 $C$ 的 Base64 字串。

2. **Homo 編碼層**：
   將 Base64 字串中的每個字元轉為 ASCII 碼，並利用**遞迴拆解演算法**，將其表示為以 $[114514, 514, 114, 14, 11, 5, 4, 1]$ 為基數的數學算式。
